# Bachelor-Project
Vores P6 bachelor/semester projekt. 

Arbejd ikke på main bracnhen. Alt arbejde skal foregå på en ny branch, som senere kan merges ind, når den er blevet godkendt af os alle. 
Lav en ny branch, hver gang du tilføjer / ændrer / fikser noget.

FØR DU FORSØGER AT MERGE NOGET IND I MAIN, SØRG FOR AT MAIN-BRANCHEN PÅ DIN COMPUTER ER DEN NYESTE VERSION!!!!

Guide:

Trin 1:

Skift til main branch, med følgende kommando: git switch main

Trin 2:

Hent den nyeste version fra github, med følgende kommando: git fetch origin

Trin 3:

Download den nyeste version af main branch, med følgende kommando: git pull origin main
